# Adaptive Maze Agent

Data-driven AI/ML maze navigation agent for the HTI technical challenge.

## Current status

This repository is developed step-by-step according to the challenge structure.

### Step 1 - Working baseline bot

In progress:
- API configuration
- Player registration
- Maze listing
- Maze entry
- Basic navigation client

Later steps:
- Data collection and analysis
- Smarter navigation strategy
- Evaluation
- Lightweight MLOps with MLflow

## Setup

Create a local `.env` file based on `.env.example`.

```bash
pip install -r requirements.txt
python -m src.main
```
